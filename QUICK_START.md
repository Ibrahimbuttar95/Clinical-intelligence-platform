# 🚀 QUICK BUILD & INSTALL COMMANDS\n\n## Step 1: Clone & Setup Backend\n```bash\ngit clone https://github.com/Ibrahimbuttar95/Clinical-intelligence-platform.git\ncd Clinical-intelligence-platform/backend\n\n# Setup virtual environment\npython -m venv venv\nsource venv/bin/activate  # Linux/Mac\n# or\nvenv\\Scripts\\activate  # Windows\n\n# Install dependencies\npip install -r requirements.txt\n\n# Create .env file\ncp .env.example .env\n# Edit .env with your settings\n\n# Run migrations (if database exists)\nalembic upgrade head\n\n# Start backend server\nuvicorn app.main:app --reload --host 0.0.0.0 --port 8000\n```\n\n**Backend is now running at:** http://localhost:8000\n\n---\n\n## Step 2: Build Flutter APK\n\n```bash\ncd ../frontend\n\n# Clean previous builds\nflutter clean\n\n# Get dependencies\nflutter pub get\n\n# Build Debug APK (for testing - ~60MB)\nflutter build apk --debug\n\n# OR Build Release APK (for production - ~30MB, optimized)\nflutter build apk --release\n```\n\n**APK Output Locations:**\n- Debug: `frontend/build/app/outputs/flutter-apk/app-debug.apk`\n- Release: `frontend/build/app/outputs/flutter-apk/app-release.apk`\n\n---\n\n## Step 3: Install APK on Android Device\n\n### Option A: Using ADB (Fastest)\n```bash\n# Enable USB Debugging on your Android phone\n# Connect phone via USB cable\n\n# List connected devices\nadb devices\n\n# Install debug APK\nadb install -r frontend/build/app/outputs/flutter-apk/app-debug.apk\n\n# Or install release APK\nadb install -r frontend/build/app/outputs/flutter-apk/app-release.apk\n\n# Launch app\nadb shell am start -n com.example.clinical_intelligence_platform/.MainActivity\n```\n\n### Option B: Manual File Installation\n1.
